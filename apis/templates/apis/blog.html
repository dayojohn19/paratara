📋 HTML Structure Guide

Required wrapper structure:

  • <article id="body-contents"> - Main article wrapper (required)
  • <article class="blog-post"> - Inner article wrapper (required)

Content CSS classes for styling:

  • .intro-section - Hero section with gradient background (blue)
  • .content-section - Main content wrapper for each section
  • .highlight-box - Light blue info box with left border
  • .tip-box - Orange tip box with left border
  • .mindset-box - Gradient blue box for special content
  • .cta-section - Orange gradient call-to-action box

⚙️ JavaScript Requirements:

  • The page automatically includes image fetching via fetchAndInsertImages()
  • Blog list sidebar populated via getBlogLists()
  • Mobile menu toggle with toggleMenu()
  • Form submission handler for image uploads
  • Bottom scroll trigger for menu display
